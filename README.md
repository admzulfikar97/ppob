# ppob
Backend Server SIMS PPOB (Java Spring Boot)

# Deskripsi Singkat

Project ini dikembangkan dengan menggunakan bahasa pemrograman Java dengan memanfaatkan framework Spring Boot.
Dimana diterapkan gaya arsitektur API (rest API) dengan design monolith modular layered structure, dimana 
berdasarkan pengalaman dan beberapa sumber yang pernah pengembang peroleh. Design monolith dengan modular structure
dapat meningkatkan skalabilitas project sehingga dapat lebih mudah untuk penambahan modul dan pemecahan menjadi
design microservice jika diperlukan.

Dalam project juga terlampir design database dalam format .png serta file json postman jika diperlukan.

Pengembang menyadari masih banyak kekurangan dalam penyelesaian project ini, karena disebabkan dikerjakan terlalu
tergesa-gesa. Tetapi pengembang berharap project ini bisa diakses tanpa adanya kendala.
